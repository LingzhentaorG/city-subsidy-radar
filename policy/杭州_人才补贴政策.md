#### 1. 杭州市应届毕业生生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 杭州市应届毕业生生活补贴 |
| **政策文号** | 杭人社发〔2023〕126号（《杭向未来·大学生创新创业三年行动计划（2023—2025年）实施细则》） |
| **适用对象** | 2019年6月3日（含）之后毕业的全球本科及以上学历全日制应届毕业生（国（境）外高校毕业生学历学位需经教育部留服中心认证）；符合条件的非全日制研究生也可参照申请 |
| **补贴金额/标准** | 本科1万元、硕士3万元、博士10万元；毕业时间在2021年10月14日（含）之前的博士，补贴标准仍为5万元。分两笔发放，每笔50% |
| **学历要求** | 全日制本科及以上 |
| **年龄限制** | null |
| **社保要求** | 第一笔：毕业后1年半内在符合条件的杭州用人单位社保连续参保（不含补缴）满6个月，且连续参保首月在毕业后1年内；第二笔：毕业后2年内在杭州用人单位社保连续参保满1年。外国国籍高校毕业生的时限放宽为毕业后6年内 |
| **申请条件** | 1.全球本科及以上学历全日制应届毕业生；2.在符合条件的杭州用人单位就业或自主创业；3.按规定缴纳社会保险；4.申请时限：中国籍毕业2年内，外国国籍毕业6年内 |
| **申请流程** | 1.申请人向用人单位提交材料；2.用人单位通过"亲清在线"数字平台或申请人在"杭州市民卡APP—杭州人才码（青荷码）"提交申请；3.系统自动比对审核；4.审核通过后拨付至申请人银行卡 |
| **申请平台** | "亲清在线"数字平台（https://qinqing.hangzhou.gov.cn）或"杭州市民卡APP—杭州人才码" |
| **申请材料** | 1.申请人姓名、证件号码、银行卡号等基本信息；2.国（境）外高校毕业生需提供教育部留学服务中心学历学位认证书编号；3.其他学历、社保信息通过系统数据共享获取；4.有劳务派遣经历者需由用工单位填写实际历史用工单位及统一社会信用代码 |
| **审核周期** | 系统自动比对审核，具体以实际拨付为准 |
| **发放方式** | 分两笔发放，每笔50%；连续参保满6个月发放第一笔，满12个月发放第二笔 |
| **生效时间** | 2019年6月3日 |
| **失效时间** | 待核实（三年行动计划有效期至2025年，后续是否延续需以官方公告为准） |
| **政策状态** | 有效（2024年5月系统已升级，支持非全日制研究生及西部区县满3年再次补贴） |
| **官方来源** | https://hrss.hangzhou.gov.cn/art/2024/5/13/art_1229810556_4336829.html |
| **最后核实日期** | 2026-06-30 |

| **申请流程** | 1.登录"浙江政务服务网"或"浙里办"APP搜索"高校毕业生就业补贴申领"；2.选择杭州市及企业注册地所在区；3.在线填写并上传材料；4.经办机构审核、公示；5.拨付 |

| **申请材料** | 1.居民身份证或社会保障卡；2.高校毕业生就业补贴申请表；3.相关学历证明（毕业证书或《教育部学籍在线验证报告》，国外学历学位认证书等） |
| **审核周期** | 30个工作日 |
| **发放方式** | 每年2000元，每连续缴纳社保满12个月申报一次 |
| **生效时间** | 2016年2月1日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效 |
